Ikut 6 langkah mudah untuk aktifkan Cloud Sync

1 Buat Firebase Project

Pergi ke Firebase Console dan buat project baru:

Buka Firebase Console ‚Üí
  • Klik "Add Project"
  • Nama project: IP WARDEN IPD YAN
  • Disable Google Analytics (optional)
  • Klik "Create Project"
2 Enable Firestore Database

Dari Firebase Console:

  • Klik "Firestore Database" di sidebar
  • Klik "Create database"
  • Pilih "Start in test mode"
  • Location: asia-southeast1 (Singapore)
  • Klik "Enable"
3 Dapatkan Firebase Config

Dari Firebase Console:

  • Klik icon gear ‚öôÔ∏è ‚Üí "Project settings"
  • Scroll ke bawah, klik icon web </>

4 Update firebase-config.js

Buka fail firebase-config.js dan gantikan:

const firebaseConfig = { apiKey: "TUKAR-DENGAN-API-KEY-ANDA", authDomain: "TUKAR-DENGAN-AUTH-DOMAIN-ANDA", projectId: "TUKAR-DENGAN-PROJECT-ID-ANDA", storageBucket: "TUKAR-DENGAN-STORAGE-BUCKET-ANDA", messagingSenderId: "TUKAR-DENGAN-MESSAGING-ID-ANDA", appId: "TUKAR-DENGAN-APP-ID-ANDA" };

Dengan config yang anda dapat dari Step 3.

5 Setup Firestore Rules (PENTING)

Dari Firestore Database, klik tab "Rules":

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{noBadan} { allow read: if true; allow write: if true; } match /siasatanRecords/{recordId} { allow read: if true; allow write: if true; } } }

Copy kod di atas, paste dalam Rules editor, dan klik "Publish"
